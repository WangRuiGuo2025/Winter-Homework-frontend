
# DEV-BLOG前端

> 基于Vue3和Element Plus的Dev-blog前端

## 项目简介

本项目用于锻炼前端项目的制作，完成寒假大作业

## 技术栈

### 核心框架/库

- **前端**：Vue3、axios、Element Plus

## 实现功能

### 核心功能

1. 实现用户注册、登录、关于、发表等页面的UI设计
2. 实现了部分基础显示逻辑
3. 实现了与后端服务器的连接

## 快速开始

### 环境要求

- 操作系统：Windows/macOS/Linux
- 运行环境：Node.js、NPM

### 安装与启动步骤

#### 1. 克隆项目

```bash
git clone https://github.com/WangRuiGuo2025/Winter-Homework-frontend.git
cd Winter-Homework-frontend
```

#### 2. 安装依赖

```bash
npm install
```

#### 3. 启动项目

```bash
npm run dev
```

#### 4. 访问项目

- 前端页面地址：http://localhost:5173
